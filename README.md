# 🐞 BugBlaster

BugBlaster is a React-based bug tracking application that allows users to create, edit, delete, and sort bug tickets based on their priority.

## 🚀 Live Demo

👉 https://kalyan565.github.io/BugBlaster

---

## ✨ Features

- 📝 Create bug tickets
- ✏️ Edit existing tickets
- 🗑️ Delete tickets
- 🔥 Assign priority (Low, Medium, High)
- 🔃 Sort tickets by:
  - High → Low
  - Low → High
- 📱 Responsive user interface

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/kalyan565/BugBlaster.git
```

Navigate to the project:

```bash
cd BugBlaster
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm start
```

Create a production build:

```bash
npm run build
```

Deploy to GitHub Pages:

```bash
npm run deploy
```
